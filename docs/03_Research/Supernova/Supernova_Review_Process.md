# Purpose

This document defines the operational review process used by Supernova.

---

# Review Levels

Monthly Review

Quarterly Review

Annual Review

---

# Monthly Review

Objective:

Monitor approved companies.

Review Items:

* Leadership status
* Competitive position
* Replacement risk
* Theme alignment

Possible Outcomes:

* Continue Accumulation
* Watch
* Review Required

---

# Quarterly Review

Objective:

Evaluate candidate universe and themes.

Review Items:

* Candidate additions
* Candidate removals
* Challenger emergence
* Theme health

Possible Outcomes:

* Universe update
* Theme update
* No Action

---

# Annual Review

Objective:

Evaluate framework effectiveness.

Review Items:

* Theme framework
* Scoring methodology
* Candidate universe quality
* Portfolio construction methodology

Possible Outcomes:

* Framework update
* No Action

---

# Review Triggers

Leader deterioration

Theme deterioration

Replacement risk escalation

Emerging challenger

---

# Review Outputs

Leadership Status

Replacement Risk

Theme Health

Recommended Actions

---

# Relationship With Portfolio

Price changes alone do not trigger review outcomes.

Leadership and theme changes are primary review drivers.
